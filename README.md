# collectie-nederland-schema-profile
This repository describes the schema profile developed for the platform [Collectie Nederland](https://collectienederland.nl/). It is a less-strict version of the [NDE Schema.org Application Profile](https://github.com/netwerk-digitaal-erfgoed/schema-profile), which is described [here](https://docs.nde.nl/schema-profile/). The shacl shape is located here: ```data/shacl.ttl```.
